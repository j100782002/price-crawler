# price-crawler


## 專案簡介
這個專案讓使用者輸入想購買的產品名稱，程式會自動從 Momo、PChome 和 Yahoo 三個購物網站爬取第一頁的搜尋結果，並將產品名稱、價格及連結抓取下來，最終上傳到 Google Sheets。這樣的工具可以幫助使用者在瀏覽購物網時避免受到其他商品的干擾，專注於最初想購買的產品。
